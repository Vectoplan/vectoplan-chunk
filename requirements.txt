# services/vectoplan-chunk/requirements.txt
# -----------------------------------------------------------------------------
# VECTOPLAN Chunk Service - Python dependencies
#
# Purpose of this service:
# - Flask microservice for project-scoped editable chunk worlds
# - PostgreSQL-backed Project / Universe / WorldInstance persistence
# - PostgreSQL-backed ChunkSnapshot persistence
# - PostgreSQL-backed WorldCommandLog / ChunkEvent history
# - preparation for future multi-block object persistence
#
# Dependency principles:
# - keep runtime dependencies small
# - use stable major-version ranges
# - prefer SQLAlchemy 2.x compatible libraries
# - use psycopg 3.x for PostgreSQL
# - keep migration support available from the start
# -----------------------------------------------------------------------------

# -----------------------------------------------------------------------------
